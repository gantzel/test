Linear
component.

The Linear component represents progress or consumption along a straight line. It is used to indicate how much of a resource has been used — such as mobile data, storage or time — and how much remains.

Three variants signal meaning at a glance: green for normal use, red for near-limit or over-threshold, and empty for a full track with no data. Optional header and footer rows add context without changing the bar itself.

03 · Anatomy

Four parts.

Header and footer are optional and independent — any combination is valid. The track and fill are always present; the fill is hidden in the Empty variant.

Data brugt i udlandet Tilkøb
5 GB brugt af 40 GB
A
Header (optional)

Flex row with justify-content: space-between. Left slot: .linear__title — H6 typography, --color-text-dark-secondary. Right slot: optional .label chip.

B
Track

Full-width bar, 8 px high. Background --color-state-tertiary-pressed (rgba 70 55 0 / 12 %), border-radius: --border-radius-4 (4 px), overflow: hidden clips the fill.

C
Fill

Width controlled by CSS custom property --linear-progress (set as an inline style). Background set by the variant modifier: --color-brand-primary (Positive) or --color-alert-error (Alert). Hidden in the Empty variant.

D
Footer (optional)

H6 typography, --color-text-dark-primary. Use to show the exact values behind the bar — e.g. "5 GB brugt af 40 GB".

04 · Tokens

Figma variables.

Fill colours come from the brand and alert token groups. The track background adapts automatically to dark mode via its existing token.

05 · Code

How to build it.

Set --linear-progress as an inline CSS custom property to control the fill width. Header and footer elements are optional — omit either when not needed.

HTML

<!-- Positive — 12.5% fill -->
<div class="linear linear--positive" style="--linear-progress: 12.5%">
  <div class="linear__header">
    <span class="linear__title">Data brugt i udlandet</span>
    <!-- optional label chip -->
    <span class="label label--neutral">Tilkøb</span>
  </div>
  <div class="linear__track">
    <div class="linear__fill"></div>
  </div>
  <div class="linear__footer">5 GB brugt af 40 GB</div>
</div>

<!-- Alert — 87.5% fill -->
<div class="linear linear--alert" style="--linear-progress: 87.5%">
  …
</div>

<!-- Empty — no fill -->
<div class="linear linear--empty">
  …
</div>

CSS

.linear {
  display: flex;
  flex-direction: column;
  gap:   var(--spacing-small);   /* 8px */
  width: 100%;
}
.linear__header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  min-height: 24px;
  gap: var(--spacing-small);
}
.linear__title {
  font-family:    var(--font-family-sans);
  font-size:      var(--type-h6-size);      /* 14px */
  font-weight:    var(--type-h6-weight);    /* 500 */
  line-height:    var(--type-h6-leading);   /* 15/14 */
  letter-spacing: var(--type-h6-tracking);  /* 0 */
  color: var(--color-text-dark-secondary);
  flex: 1;
}
.linear__track {
  height:        var(--spacing-small);          /* 8px */
  border-radius: var(--border-radius-4);         /* 4px */
  background:    var(--color-state-tertiary-pressed);
  overflow: hidden;
  width: 100%;
}
.linear__fill {
  height: 100%;
  width:  var(--linear-progress, 0%);
  border-radius: var(--border-radius-4);
  transition: width 0.4s ease;
}
.linear__footer {
  font-family:    var(--font-family-sans);
  font-size:      var(--type-h6-size);
  font-weight:    var(--type-h6-weight);
  line-height:    var(--type-h6-leading);
  letter-spacing: var(--type-h6-tracking);
  color: var(--color-text-dark-primary);
}

.linear--positive .linear__fill { background: var(--color-brand-primary); }
.linear--alert    .linear__fill { background: var(--color-alert-error); }
.linear--empty    .linear__fill { display: none; }

/* Dark mode */
[data-color-theme="dark"] .linear__title  { color: var(--color-text-light-secondary); }
[data-color-theme="dark"] .linear__footer { color: var(--color-text-light-primary); }
