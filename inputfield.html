Input Field
component.

The Input Field is a single-line text control used to collect user input. It features a floating label that starts inside the field and rises to sit on the border when the field is focused or filled.

Two variants cover the main use cases: Text for free-form input and Number for phone numbers with a fixed country-code prefix. An optional search icon can be placed on the right.

03 · Anatomy

Four parts.

B
A
Wrapper

The bordered container. min-height: 48px, border-radius: --border-radius-8 (8 px). Default border is 1 px --color-border-default; focused or selected state uses 2 px --color-border-active. In dark mode the active border becomes brand green.

B
Input value

The actual <input> element. H5 Medium — 16 px / weight 500. Colour: --color-text-dark-primary. Uses placeholder=" " (single space) so CSS :not(:placeholder-shown) can detect when the field is filled.

C
Floating label

Positioned absolute, starts centered vertically at H5 size. On focus or when the field has a value, it transitions to top: 0 at 12 px / 0.3 px tracking, sitting on the border line with background: --color-bg-frame to visually cut the border.

D
Icon button

Optional search icon, 20 × 20 px, inside a 32 × 32 px round tap target with 6 px padding. Omit the element entirely when no icon is needed.

05 · Code

How to build it.

Use placeholder=" " (a single space) on the <input> so the CSS :not(:placeholder-shown) selector can detect when the field has a value and float the label automatically.

Text variant — HTML

<!-- Default / empty -->
<div class="input-field">
  <div class="input-field__wrapper">
    <input class="input-field__input" type="text"
           id="f1" placeholder=" " />
    <label class="input-field__label" for="f1">Label Text</label>
  </div>
</div>

<!-- With icon right -->
<div class="input-field">
  <div class="input-field__wrapper">
    <input class="input-field__input" type="text"
           id="f2" placeholder=" " />
    <label class="input-field__label" for="f2">Label Text</label>
    <button class="input-field__icon-btn" type="button"
            aria-label="Search">
      <!-- 20×20 search SVG -->
    </button>
  </div>
</div>

<!-- With helper text -->
<div class="input-field input-field--error">
  <div class="input-field__wrapper">
    <input class="input-field__input" type="text"
           id="f3" placeholder=" "
           aria-describedby="f3-helper" aria-invalid="true" />
    <label class="input-field__label" for="f3">Label Text</label>
  </div>
  <p class="input-field__helper" id="f3-helper">Helper text</p>
</div>

Number variant — HTML

<div class="input-field input-field--number">
  <div class="input-field__wrapper">
    <span class="input-field__prefix">+45</span>
    <input class="input-field__input" type="tel"
           id="n1" placeholder=" " />
    <label class="input-field__label" for="n1">Label Text</label>
  </div>
</div>

State modifiers

/* Applied to .input-field */
.input-field--error     /* Red border + red helper text */
.input-field--disabled  /* All text to --color-disabled, pointer-events: none */
.input-field--number    /* Enables prefix slot and tel input */
.input-field--selected  /* Static active state (2px border, floated label) */
