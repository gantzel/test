Label
component.

The eCare label is a compact status badge used to communicate category, state or priority at a glance. It appears on product cards, list items and inline with content to signal meaning without taking up space.

Six semantic variants cover the full feedback range — from product promotion to error and disabled states. The pill shape, fixed H7 typography and tight padding are all defined by Figma tokens and do not change across breakpoints.

02 · Examples

Six variants.

Each variant maps to a semantic use case. Use Extra for product highlights, the three feedback variants for status, Neutral for general information, and Disabled when interaction is unavailable.

03 · Anatomy

Two parts.

The label is a single-level component. The container defines all visual properties — the text slot receives whatever string is passed in.

Populær
A
Container

Sets background to the variant color token, border-radius to --border-radius-round (512 px pill) and padding to --spacing-x-small × --spacing-small (4 × 8 px). Uses inline-flex so width is always content-driven.

B
Text

H7 typography — 12 px / 14 px line-height / weight 500. Text color is fixed per variant and does not inherit from context. Keep labels short: one to two words maximum.

04 · Tokens

Figma variables.

Each variant has its own background token from the Feedback & Label group. Shape, padding and typography are shared tokens constant across all variants and breakpoints.

05 · Code

How to build it.

An inline <span class="label label--[variant]">. The variant modifier sets the background — all other properties come from the base class.

HTML

<span class="label label--extra">Populær</span>
<span class="label label--neutral">Information</span>
<span class="label label--warning">Warning</span>
<span class="label label--error">Error</span>
<span class="label label--success">Success</span>
<span class="label label--disabled">Disabled</span>

CSS

.label {
  display: inline-flex;
  align-items: center;
  padding:        var(--spacing-x-small) var(--spacing-small);  /* 4px 8px */
  border-radius:  var(--border-radius-round);                    /* 512px — pill */
  font-family:    var(--type-h7-family);
  font-size:      var(--type-h7-size);                           /* 12px */
  font-weight:    var(--type-h7-weight);                         /* 500 */
  line-height:    var(--type-h7-leading);                        /* 14px */
  letter-spacing: var(--type-h7-tracking);
  white-space: nowrap;
}

.label--extra    { background: var(--color-label-extra);   color: var(--color-text-light-primary); }
.label--neutral  { background: var(--color-label-general); color: var(--color-text-dark-primary); }
.label--warning  { background: var(--color-label-warning); color: var(--color-text-always-black-p); }
.label--error    { background: var(--color-label-error);   color: var(--color-text-always-black-p); }
.label--success  { background: var(--color-label-success); color: var(--color-text-always-black-p); }
.label--disabled { background: var(--color-disabled-bg);   color: var(--color-disabled); }

/* Dark mode: bg flips on extra and neutral — swap text to stay readable */
[data-color-theme="dark"] .label--extra   { color: var(--color-text-always-black-p); }
[data-color-theme="dark"] .label--neutral { color: var(--color-text-light-primary); }
