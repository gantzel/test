Radio Button
component.

The Radio Button is a single-selection control used when the user must choose exactly one option from a set. It consists of a circular hit area containing a 24 px icon — an outlined ring that fills with an inner dot when selected.

The control is always used in a group of two or more options sharing the same name attribute. Always pair the radio input with a visible text label so the choice is clear to all users.

02 · States

Eight states.

Four interaction states (Normal, Hover, Pressed, Disabled) combined with two selection states (Unselected, Selected) give eight visual states in total. The tertiary overlay tokens drive the hover and pressed backgrounds.

03 · Anatomy

Three parts.

C
A
Hit area

The outermost element — 32 × 32 px (4 px padding around the 24 px icon). border-radius: --border-radius-round (512 px) creates the circular tap target. Receives the hover and pressed background overlays.

B
Outer ring

A 20 × 20 px circle with a 1.5 px border, inset 2 px from the 24 px icon container. Colour is --color-text-dark-primary in normal state, --color-disabled when inactive.

C
Inner dot

A 10 × 10 px filled circle centred inside the icon (7 px from each edge). Visible only in the selected state. Uses the same colour token as the outer ring.

04 · Tokens

Figma variables.

The ring and dot colour adapts between light and dark mode via dedicated text tokens. State overlays and the disabled token are the same as other interactive components.

05 · Code

How to build it.

Wrap a visually-hidden native <input type="radio"> and a .radio__icon span inside a <label>. The CSS sibling combinator (~) drives the selected state without JavaScript. All radios in a group share the same name.

HTML structure

<!-- A full radio group -->
<fieldset>
  <legend>Choose an option</legend>

  <label class="radio-label">
    <span class="radio">
      <input type="radio" class="radio__native" name="group" value="a" />
      <span class="radio__icon" aria-hidden="true"></span>
    </span>
    Option A
  </label>

  <label class="radio-label">
    <span class="radio">
      <input type="radio" class="radio__native" name="group" value="b" checked />
      <span class="radio__icon" aria-hidden="true"></span>
    </span>
    Option B
  </label>

  <label class="radio-label radio-label--inactive">
    <span class="radio radio--inactive">
      <input type="radio" class="radio__native" name="group" value="c" disabled />
      <span class="radio__icon" aria-hidden="true"></span>
    </span>
    Option C
  </label>
</fieldset>

Modifiers

/* On .radio */
.radio--selected   /* shows inner dot — use when not wrapping a native input */
.radio--inactive   /* disabled appearance + pointer-events: none */

/* On .radio-label */
.radio-label--inactive  /* dims the label text alongside the control */

CSS

.radio {
  display: inline-flex;
  align-items: center;
  padding: 4px;
  border-radius: var(--border-radius-round);
  border: none;
  background: transparent;
  cursor: pointer;
  transition: background 0.15s ease;
}
.radio:hover  { background: var(--color-state-tertiary-hover); }
.radio:active { background: var(--color-state-tertiary-pressed); }

.radio__native { position: absolute; opacity: 0; width: 0; height: 0; }

.radio__icon::before {              /* outer ring */
  content: ''; position: absolute;
  top: 2px; left: 2px; width: 20px; height: 20px;
  border-radius: 50%;
  border: 1.5px solid var(--color-text-dark-primary);
}
.radio__native:checked ~ .radio__icon::after {   /* inner dot */
  content: ''; position: absolute;
  top: 7px; left: 7px; width: 10px; height: 10px;
  border-radius: 50%;
  background: var(--color-text-dark-primary);
}
.radio--inactive .radio__icon::before { border-color: var(--color-disabled); }
.radio--inactive.radio--selected .radio__icon::after { background: var(--color-disabled); }
