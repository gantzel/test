Status
component.

The Status component communicates the current state of a product, subscription or process using a small colour-coded dot paired with a short text label. It provides an immediate visual signal without requiring the user to read long descriptions.

Six dot colours cover the full signal range — active, informational, warning, error, neutral and disabled. Dot size, gap and typography are all driven by shared tokens and stay constant across breakpoints.

03 · Anatomy

Two parts.

The Status component is a horizontal flex row with a fixed-size dot and a free-form text label. Only the dot colour changes between variants.

Aktivering igang
A
Dot

An 8 × 8 px circle — width and height both set to --spacing-small (8 px), border-radius to --border-radius-round (512 px). Background colour is set by the variant modifier class. flex-shrink: 0 prevents it from collapsing.

B
Text

Label typography — 12 px / weight 500 / line-height 1 / tracking 0.003 em. Colour is --color-text-dark-primary in light mode, --color-text-light-primary in dark mode. Use short, descriptive state text — one to four words.

04 · Tokens

Figma variables.

The four system status colours are constant across themes. The neutral dot and text colour adapt to the active mode via dark-mode overrides.

Token Figma variable Light Dark
--color-sys-success System/System Colors/100% Success --color-sys-success — constant
--color-sys-info System/System Colors/100% Info --color-sys-info — constant
--color-sys-warning System/System Colors/100% Warning --color-sys-warning — constant
--color-sys-error System/System Colors/100% Error --color-sys-error — constant
--color-text-dark-primary Text/Dark Colors/Text Primary --color-text-dark-primary --color-text-light-primary
--color-disabled-bg System/Disabled/Disabled Background --color-disabled-bg --color-disabled-bg
--spacing-small Spacing/Small 8 px — dot size & gap, constant
--border-radius-round Border/Border Radius/Round 512 px — constant
Label typography Font/Components/Label 12 px · weight 500 · line-height 1 · tracking 0.003 em — constant
05 · Code

How to build it.

A <span class="status status--[variant]"> wrapper containing a dot span and a text span. The variant modifier sets only the dot colour.

HTML

<span class="status status--success">
  <span class="status__dot"></span>
  <span class="status__text">Aktiv</span>
</span>

<!-- Other variants -->
<span class="status status--info"> … </span>
<span class="status status--warning"> … </span>
<span class="status status--error"> … </span>
<span class="status status--neutral"> … </span>
<span class="status status--disabled"> … </span>

CSS

.status {
  display: inline-flex;
  align-items: center;
  gap: var(--spacing-small);    /* 8px */
}
.status__dot {
  width:  var(--spacing-small);  /* 8px */
  height: var(--spacing-small);
  border-radius: var(--border-radius-round);
  flex-shrink: 0;
}
.status__text {
  font-family:    var(--font-family-sans);
  font-size:      var(--type-label-size);     /* 12px */
  font-weight:    var(--type-label-weight);   /* 500 */
  line-height:    var(--type-label-leading);  /* 1 */
  letter-spacing: var(--type-label-tracking); /* 0.003em */
  color: var(--color-text-dark-primary);
  white-space: nowrap;
}

.status--success  .status__dot { background: var(--color-sys-success); }
.status--info     .status__dot { background: var(--color-sys-info); }
.status--warning  .status__dot { background: var(--color-sys-warning); }
.status--error    .status__dot { background: var(--color-sys-error); }
.status--neutral  .status__dot { background: var(--color-text-dark-primary); }
.status--disabled .status__dot { background: var(--color-disabled-bg); }

/* Dark mode */
[data-color-theme="dark"] .status__text {
  color: var(--color-text-light-primary);
}
[data-color-theme="dark"] .status--neutral .status__dot {
  background: var(--color-text-light-primary);
}
