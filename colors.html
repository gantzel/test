Colors
foundation.

The eCare color system is built directly from Figma variables and exported as CSS custom properties. Every token has a semantic name that describes its purpose, so the system stays meaningful as the palette evolves.

The system covers two themes — Light and Dark — with 68 tokens per theme. Use the toggle at the top of each section to preview how colors shift between modes.

10 · Code

How to use it.

Import tokens.css and reference the CSS custom properties. The file defines :root for Light theme and [data-theme="dark"] overrides for Dark theme.

CSS tokens (excerpt)

/* eCare Design System — Color tokens
   Source: Figma variables · Light & Dark Theme
   Exported: May 2026 */

:root {
  /* Brand */
  --color-brand-primary:   #00DC00;
  --color-brand-secondary: #323232;

  /* Alerts */
  --color-alert-success:   #63E65F;
  --color-alert-info:      #CFD4E2;
  --color-alert-warning:   #F8DFB2;
  --color-alert-error:     #FB8F82;

  /* Text */
  --color-text-dark-primary:   rgba(0, 0, 0, 0.8);
  --color-text-dark-secondary: rgba(0, 0, 0, 0.56);

  /* … 62 more tokens */
}

[data-theme="dark"] {
  --color-brand-secondary: #FFFFFF;
  --color-alert-success:   #02C801;
  --color-alert-info:      #4B5B7E;
  --color-alert-warning:   #E6B967;
  --color-alert-error:     #E86A5A;
  /* … dark overrides */
}

Usage examples

/* Alert component */
.alert--success {
  background: var(--color-alert-success);
  color:      var(--color-text-dark-primary);
}

/* Dark-mode button */
.btn-primary {
  background: var(--color-brand-primary);
  color:      var(--color-text-dark-primary);
}
.btn-primary:hover {
  background: var(--color-state-primary-hover);
}

/* Disabled state */
.btn:disabled {
  background: var(--color-disabled-bg);
  color:      var(--color-disabled);
}

Load order

<!-- Load tokens first -->
<link rel="stylesheet" href="css/tokens.css">
<link rel="stylesheet" href="css/typography.css">

<!-- Set theme on root element -->
<body data-theme="dark"> // dark mode
<body>                  // light mode (default)
