Radius
foundation.

The eCare radius scale defines how rounded element corners appear across the system. Every step is a constant CSS custom property — the same value in both light and dark mode — and maps directly to a named Figma Border Radius variable.

Five steps cover the full range: from sharp square corners (None, 0 px) through small and medium increments suited to inputs and buttons, to the pill radius used for labels, status dots and progress indicators (Round, 512 px).

03 · Tokens

Token reference.

All five radius values are constant — they do not change between light and dark mode.

Token Figma variable Value Used in
--border-radius-none Border / Border Radius / None 0 px Sharp-cornered elements with no rounding
--border-radius-4 Border / Border Radius / 4 4 px Linear progress track · small inputs
--border-radius-8 Border / Border Radius / 8 8 px Buttons · medium-sized UI elements
