Button
component.

The Button is the primary call-to-action element in eCare. It comes in three styles — Primary, Secondary, and Outlined — each signalling a different level of emphasis. All three share the same pill shape and typographic scale.

Two sizes (Large 32 px and Small 28 px) cover the full range of contexts from hero actions to compact inline controls. A dedicated Loading state replaces the label with a spinner while an action completes.

02 · Styles

Three styles.

Each style communicates a different level of emphasis. Use Primary for the single most important action, Secondary for supporting actions, and Outlined for low-emphasis or destructive alternatives.

Primary

High emphasis. Green fill with always-black text. Reserved for the single primary CTA per view.

.btn--primary
Secondary

Medium emphasis. Dark fill with light text in light mode; inverts to white in dark mode. Use alongside a Primary action.

.btn--secondary
Outlined

Low emphasis. Transparent background with 2 px border. Use for tertiary actions or where a filled button would dominate.

.btn--outlined
03 · States

Five states.

All three styles support the same five states. Hover and Pressed use overlay colours from the token system. Inactive disables interaction and reduces contrast. Loading replaces the label with an animated spinner.

04 · Anatomy

Two parts.

A
Container

The button element itself. Height 32 px (Large) or 28 px (Small). Horizontal padding --spacing-medium (16 px). border-radius: --border-radius-round (512 px) produces the pill shape. Background and border depend on the active style.

B
Label

Button Large uses --type-button-large-size (16 px), Button Small uses --type-button-medium-size (14 px). Both are --font-weight-medium (500) with --letter-spacing-button (0.2 px). In the Loading state the label is replaced by .btn__spinner.

06 · Code

How to build it.

Compose two BEM modifiers on a single element — one for style (.btn--primary, .btn--secondary, .btn--outlined) and one for size (.btn--large, .btn--small). The element can be a <button> or an <a> tag interchangeably.

HTML structure

<!-- Style + size modifiers on a single element -->
<button class="btn btn--primary btn--large">Button</button>

<!-- Works as an anchor too -->
<a class="btn btn--primary btn--large" href="#">Button</a>

<!-- Loading state — replace label with spinner -->
<button class="btn btn--primary btn--large btn--loading">
  <span class="btn__spinner" aria-hidden="true"></span>
</button>

<!-- Inactive / disabled -->
<button class="btn btn--primary btn--large btn--inactive" disabled>Button</button>

Modifiers

/* Style — pick one */
.btn--primary    /* filled green · primary action */
.btn--secondary  /* filled grey  · secondary action */
.btn--outlined   /* border only  · tertiary / ghost */

/* Size — pick one */
.btn--large      /* 32px height · default touch target */
.btn--small      /* 28px height · compact contexts */

/* State — add as needed */
.btn--inactive   /* disabled appearance + pointer-events: none */
.btn--loading    /* hides cursor, pairs with .btn__spinner child */

CSS

.btn {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  border-radius: var(--border-radius-round);   /* pill shape */
  padding: 0 var(--spacing-medium);
  font-weight: var(--font-weight-medium);        /* 500 */
  border: none;
  cursor: pointer;
  transition: background 0.15s ease, border-color 0.15s ease, color 0.15s ease;
}

.btn--large { height: 32px; font-size: var(--type-button-large-size); }
.btn--small { height: 28px; font-size: var(--type-button-medium-size); }

.btn--primary   { background: var(--color-brand-primary);   color: var(--color-text-always-black-p); }
.btn--secondary { background: var(--color-state-button);    color: var(--color-text-light-primary); }
.btn--outlined  { background: transparent; border: 2px solid var(--color-text-dark-primary); }

.btn--primary:hover    { background: var(--color-state-primary-hover); }
.btn--secondary:hover  { background: var(--color-state-secondary-hover); }
.btn--outlined:hover   { background: var(--color-state-tertiary-hover); }

.btn--inactive, .btn:disabled {
  background: var(--color-disabled-bg);
  color: var(--color-disabled);
  cursor: not-allowed;
  pointer-events: none;
}

.btn__spinner {
  width: 20px; height: 20px;
  border: 2px solid currentColor;
  border-top-color: transparent;
  border-radius: var(--border-radius-round);
  animation: btn-spin 0.7s linear infinite;
}
@keyframes btn-spin { to { transform: rotate(360deg); } }
