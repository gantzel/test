Checkbox
component.

The Checkbox is a binary selection control used when the user can independently toggle one or more options. It consists of a circular hit area containing a 24 px icon — a rounded square outline that fills with a checkmark when selected.

Unlike radio buttons, checkboxes in a group are independent — any number can be selected simultaneously. Always pair each checkbox with a visible text label to make the option clear to all users.

02 · States

Eight states.

Four interaction states (Normal, Hover, Pressed, Disabled) combined with two selection states (Unselected, Selected) give eight visual states in total. The tertiary overlay tokens drive the hover and pressed backgrounds.

03 · Anatomy

Three parts.

C
A
Hit area

The outermost element — 32 × 32 px (4 px padding around the 24 px icon). border-radius: --border-radius-round (512 px) creates the circular tap target. Receives the hover and pressed background overlays.

B
Box

An 18 × 18 px rounded square inset 3 px from the 24 px icon container. Corner radius is --border-radius-4 (4 px). In the unselected state it shows a 2 px border; in the selected state the interior fills with --color-text-dark-primary.

C
Checkmark

A CSS L-shape (border-left + border-bottom, rotated −45°) centred inside the box. Visible only in the selected state. Uses --color-fixed-white in light mode and --color-fixed-black in dark mode to always contrast with the filled box.

04 · Tokens

Figma variables.

The box border and fill adapt between light and dark mode via the text-primary tokens. The checkmark always uses a fixed colour that contrasts with the filled box.

05 · Code

How to build it.

Wrap a visually-hidden native <input type="checkbox"> and a .checkbox__icon span inside a <label>. The CSS sibling combinator (~) drives the selected state without JavaScript. Multiple checkboxes in a group are independent — no shared name grouping is required.

HTML structure

<!-- A group of independent checkboxes -->
<fieldset>
  <legend>Choose options</legend>

  <label class="checkbox-label">
    <span class="checkbox">
      <input type="checkbox" class="checkbox__native" name="option-a" />
      <span class="checkbox__icon" aria-hidden="true">
        <svg class="checkbox__check" viewBox="0 0 12 12"><use href="#icon-check"/></svg>
      </span>
    </span>
    Option A
  </label>

  <label class="checkbox-label">
    <span class="checkbox">
      <input type="checkbox" class="checkbox__native" name="option-b" checked />
      <span class="checkbox__icon" aria-hidden="true">
        <svg class="checkbox__check" viewBox="0 0 12 12"><use href="#icon-check"/></svg>
      </span>
    </span>
    Option B
  </label>

  <label class="checkbox-label checkbox-label--inactive">
    <span class="checkbox checkbox--inactive">
      <input type="checkbox" class="checkbox__native" name="option-c" disabled />
      <span class="checkbox__icon" aria-hidden="true">
        <svg class="checkbox__check" viewBox="0 0 12 12"><use href="#icon-check"/></svg>
      </span>
    </span>
    Option C
  </label>
</fieldset>

Modifiers

/* On .checkbox */
.checkbox--selected   /* shows filled box + checkmark — use without native input */
.checkbox--inactive   /* disabled appearance + pointer-events: none */

/* On .checkbox-label */
.checkbox-label--inactive  /* dims the label text alongside the control */

CSS

.checkbox {
  display: inline-flex;
  align-items: center;
  padding: 4px;
  border-radius: var(--border-radius-round);
  border: none;
  background: transparent;
  cursor: pointer;
  transition: background 0.15s ease;
}
.checkbox:hover  { background: var(--color-state-tertiary-hover); }
.checkbox:active { background: var(--color-state-tertiary-pressed); }

.checkbox__native { position: absolute; opacity: 0; width: 0; height: 0; }

.checkbox__icon::before {              /* box outline */
  content: ''; position: absolute;
  top: 3px; left: 3px; width: 18px; height: 18px;
  border-radius: var(--border-radius-4);
  border: 2px solid var(--color-text-dark-primary);
}
.checkbox--selected .checkbox__icon::before {  /* filled box — no border */
  background: var(--color-text-dark-primary);
  border: none;
}
.checkbox__check {                          /* checkmark icon — hidden by default */
  position: absolute; left: 6px; top: 6px;
  width: 12px; height: 12px;
  display: none;
  color: var(--color-fixed-white);
}
.checkbox--selected .checkbox__check { display: block; }
/* dark mode: flip checkmark to black on the light-filled box */
[data-color-theme="dark"] .checkbox--selected .checkbox__check {
  color: var(--color-fixed-black);
}
.checkbox--inactive .checkbox__icon::before { border-color: var(--color-disabled); }
.checkbox--inactive.checkbox--selected .checkbox__icon::before {
  background: var(--color-disabled);
  border-color: var(--color-disabled);
}
