Spinner
component.

The Spinner is an animated loading indicator that communicates that an operation is in progress. It is a rotating arc rendered entirely in CSS — no image assets required.

Two colour variants cover all contexts: Primary (brand green) for use on light surfaces and Secondary (adaptive dark/light) for use on any surface where the green would be inappropriate.

02 · Variants

Two variants.

Primary uses the brand green and is the default choice on white and light grey surfaces. Secondary adapts to the current theme — dark on light, light on dark.

Primary spinner
Default — brand green on light surfaces
Secondary spinner--secondary
Adaptive — dark in light mode, light in dark mode
03 · Anatomy

One part.

A
Arc

A single 40 × 40 px element with border-radius: 50% and a 4 px border. Three sides are coloured; one side is transparent to form the open arc. A CSS @keyframes animation rotates it continuously at 0.75 s per revolution.

04 · Tokens

Figma variables.

The Primary variant is always brand green. The Secondary variant follows the text colour of the current theme so it always contrasts with the surface it sits on.

Token Figma variable Light Dark
--color-brand-primary Brand / Primary Primary arc — constant
--color-text-dark-primary Text / Dark Colors / Text Primary Secondary arc — not used
--color-text-light-primary Text / Light Colors / Text Primary — not used Secondary arc
05 · Code

How to build it.

A single div with role="status" and an aria-label for screen readers. The modifier class switches the colour variant.

HTML structure

<!-- Primary (brand green) -->
<div class="spinner"
     role="status"
     aria-label="Loading"></div>

<!-- Secondary (adaptive) -->
<div class="spinner spinner--secondary"
     role="status"
     aria-label="Loading"></div>

Modifier

/* Applied to .spinner */
.spinner--secondary  /* Dark in light mode, light in dark mode */

CSS

.spinner {
  display:       inline-block;
  width:         40px;
  height:        40px;
  border-radius: 50%;
  border:        4px solid transparent;
  border-top-color:    var(--color-brand-primary);
  border-right-color:  var(--color-brand-primary);
  border-bottom-color: var(--color-brand-primary);
  animation:     spinner-rotate 0.75s linear infinite;
}

.spinner--secondary {
  border-top-color:    var(--color-text-dark-primary);
  border-right-color:  var(--color-text-dark-primary);
  border-bottom-color: var(--color-text-dark-primary);
}

[data-color-theme="dark"] .spinner--secondary {
  border-top-color:    var(--color-text-light-primary);
  border-right-color:  var(--color-text-light-primary);
  border-bottom-color: var(--color-text-light-primary);
}

@keyframes spinner-rotate {
  to { transform: rotate(360deg); }
}
