Tabs
component.

The Tab is a pill-shaped toggle button that switches between views or filters content. In its standalone form it acts as a single toggle; grouped inside a .tab-group it becomes a segmented control where exactly one tab is active at a time.

The active state fills the pill with --color-brand-secondary and flips the text to light. The group container provides its own warm-grey background so individual tabs inside are transparent when inactive.

03 · Anatomy

Three parts.

C
A
Group container

A pill-shaped wrapper with background: --color-state-filled-warm and border-radius: --border-radius-round. Items are spaced 2 px apart via gap. Not present in the standalone variant.

B
Active tab

Filled with --color-brand-secondary (#323232 light / #FFFFFF dark) and pill-shaped. Text uses --color-text-light-primary in light mode and --color-text-dark-primary in dark mode to always contrast with the fill. Padding: 9 × 14 px, min-width: 60 px.

C
Inactive tab

Transparent inside a group — the container's warm-grey shows through. As a standalone tab it carries --color-state-filled-warm as its own background. Text is --color-text-dark-primary.

04 · Tokens

Figma variables.

Active and inactive colours adapt automatically to dark mode through semantic tokens — no explicit overrides required except for the active tab text, which must flip when the background goes white.

Token Figma variable Light Dark
--color-brand-secondary Brand / Secondary Active tab bg Active tab bg
--color-state-filled-warm System / State Overlays / Filled Warm Grey Inactive bg · group bg Inactive bg · group bg
--color-text-light-primary Text / Light Colors / Text Primary Active tab text — see below
--color-text-dark-primary Text / Dark Colors / Text Primary Inactive tab text Active tab text
--border-radius-round Border / Border Radius / Round 512 px — constant
05 · Code

How to build it.

A standalone .tab is a plain <button>. In a group, wrap the tabs in .tab-group with role="tablist" and toggle .tab--active and aria-selected via JS on click.

HTML — Standalone tab

<!-- Active -->
<button class="tab tab--active" role="tab" aria-selected="true">Label</button>

<!-- Inactive -->
<button class="tab" role="tab" aria-selected="false">Label</button>

HTML — Tab group

<div class="tab-group" role="tablist" aria-label="View">
  <button class="tab tab--active" role="tab" aria-selected="true">Day</button>
  <button class="tab"            role="tab" aria-selected="false">Week</button>
  <button class="tab"            role="tab" aria-selected="false">Month</button>
</div>

JavaScript

// Tab group — one active at a time
document.querySelectorAll('.tab-group').forEach(group => {
  group.querySelectorAll('.tab').forEach(tab => {
    tab.addEventListener('click', () => {
      group.querySelectorAll('.tab').forEach(t => {
        t.classList.remove('tab--active');
        t.setAttribute('aria-selected', 'false');
      });
      tab.classList.add('tab--active');
      tab.setAttribute('aria-selected', 'true');
    });
  });
});

  cursor: pointer;

  background: var(--color-brand-secondary);
  color: var(--color-text-light-primary);
}
/* Dark mode: active becomes white pill → text must be dark */
[data-color-theme="dark"] .tab--active { color: var(--color-text-dark-primary); }

.tab-group {
  display: inline-flex; gap: 2px;
  background: var(--color-state-filled-warm);
  border-radius: var(--border-radius-round);
}
.tab-group .tab          { background: transparent; }
.tab-group .tab--active  { background: var(--color-brand-secondary); }
