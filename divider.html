Divider
component.

The Divider is a single 1 px horizontal line that creates a visual separation between adjacent content blocks — list rows, sections, or groups of related elements. It carries no semantic meaning beyond structure, so it is marked as presentational.

A single token, --color-divider, drives the colour in both light and dark mode. The line is always 1 px tall and stretches to fill its container. No variants, no modifiers.

02 · Usage

In context.

Place a Divider between list rows, between a header and its content, or to mark a transition between two content groups. Avoid stacking dividers or using them as decorative elements.

03 · Anatomy

One part.

A
A
Line

A block-level element, 1 px tall, stretching to the full width of its container. Background colour is --color-divider — an rgba value at 23% opacity that adapts automatically to light and dark mode. No border, no padding, no margin.

04 · Tokens

Figma variables.

A single adaptive token controls the line colour. It resolves to a different rgba value in light and dark mode, so the visual weight stays consistent across themes.

05 · Code

How to build it.

A single <hr> element with the .divider class. Mark it aria-hidden="true" when it is purely decorative — which is almost always the case.

HTML

<!-- Standard usage — decorative, hidden from screen readers -->
<hr class="divider" aria-hidden="true" />

<!-- Semantic separator — omit aria-hidden if the separation is meaningful -->
<hr class="divider" role="separator" />

CSS

.divider {
  display: block;
  height: 1px;
  background: var(--color-divider);
  border: none;
  width: 100%;
  margin: 0;
}
