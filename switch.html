Switch
component.

The Switch is a binary toggle control that immediately applies a setting when tapped — no confirmation needed. It consists of a 40 × 18 px pill track and a 14 × 14 px thumb that slides between the off (left) and on (right) positions.

Use a Switch for settings that take effect instantly, such as enabling notifications or toggling a feature. For choices that require explicit submission, use a Checkbox instead. Always pair the Switch with a visible label.

02 · States

Eight states.

Four interaction states (Normal, Hover, Pressed, Disabled) combined with two selection states (Off, On) give eight visual states in total. The hover and pressed circles follow the thumb position via a CSS transition on the pseudo-element.

03 · Anatomy

Two parts.

A
Track

A 40 × 18 px pill shape with border-radius: --border-radius-round (512 px). In the off state the fill is --color-disabled; in the on state it becomes --color-brand-secondary. The hover and pressed circle (32 × 32 px) is a ::before pseudo-element that transitions with the thumb.

B
Thumb

A 14 × 14 px circle inset 2 px from the track edges. Its fill is --color-bg-frame, which resolves to white in light mode and #323232 in dark mode — always contrasting with the track without any dark mode override. Slides from left: 2px (off) to left: 24px (on).

04 · Tokens

Figma variables.

Track and thumb colours adapt to light and dark mode automatically through semantic tokens — no explicit dark mode overrides are needed in the component CSS.

--color-bg-frame Background / Frame Thumb fill Thumb fill
--color-disabled-switch System / Disabled / Switch Track disabled OFF Track disabled OFF

05 · Code

How to build it.

Use a <button role="switch"> so keyboard and assistive technology users can toggle the state. The .switch--selected modifier and aria-checked must be toggled together via a small JavaScript click handler.

HTML structure

<!-- Off state -->
<button class="switch"
        role="switch"
        aria-checked="false"
        aria-label="Notifications">
  <span class="switch__track">
    <span class="switch__thumb"></span>
  </span>
</button>

<!-- On state -->
<button class="switch switch--selected"
        role="switch"
        aria-checked="true"
        aria-label="Notifications">
  <span class="switch__track">
    <span class="switch__thumb"></span>
  </span>
</button>

<!-- Disabled -->
<button class="switch switch--inactive"
        role="switch"
        aria-checked="false"
        aria-disabled="true"
        disabled>
  <span class="switch__track">
    <span class="switch__thumb"></span>
  </span>
</button>

JavaScript toggle

document.querySelectorAll('.switch:not(.switch--inactive)').forEach(btn => {
  btn.addEventListener('click', () => {
    const on = btn.classList.toggle('switch--selected');
    btn.setAttribute('aria-checked', on);
  });
});

Modifiers

/* On .switch */
.switch--selected   /* on state — slides thumb right, fills track */
.switch--inactive   /* disabled appearance + pointer-events: none */

CSS

.switch {
  display: inline-flex;
  align-items: center;
  padding: 7px;
  border-radius: var(--border-radius-round);
  border: none;
  background: transparent;
  cursor: pointer;

.switch__track {
  position: relative;
  width: 40px; height: 18px;
  border-radius: var(--border-radius-round);
  background: var(--color-disabled);
  transition: background 0.2s ease;
}
/* hover/pressed circle — follows thumb */
.switch__track::before {
  content: ''; position: absolute;
  width: 32px; height: 32px;
  border-radius: var(--border-radius-round);
  top: -7px; left: -7px;
  transition: left 0.2s ease, background 0.15s ease;
  pointer-events: none; z-index: 0;
}
.switch--selected .switch__track::before  { left: 15px; }
.switch:hover .switch__track::before      { background: var(--color-state-tertiary-hover); }
.switch:active .switch__track::before     { background: var(--color-state-tertiary-pressed); }
.switch--selected .switch__track          { background: var(--color-brand-secondary); }
.switch__thumb {
  position: absolute;
  top: 2px; left: 2px;
  width: 14px; height: 14px;
  border-radius: var(--border-radius-round);
  background: var(--color-bg-frame);
  transition: left 0.2s ease;
  z-index: 1;
}
.switch--selected .switch__thumb          { left: 24px; }
.switch--inactive                         { cursor: not-allowed; pointer-events: none; }
.switch--inactive .switch__track          { background: var(--color-disabled-switch); }
.switch--inactive.switch--selected .switch__track { background: var(--color-disabled); }
